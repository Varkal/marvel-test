What is this ?

This website is a graphical interface to vizualize a part of the datas provided by the Marvel Comics API.
For the moment, you can browse comic book list and comic book creator list, and see some information about it and filter them with ome basics criterias like name or title

What are the stack ?

This website has been created with AngularJS, Bootstrap, Angular-Ui, Lodash and Spinkit

Where are the sources ?

The source code of this website is MIT licensed and visible and downloadable on Github
